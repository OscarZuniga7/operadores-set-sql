Esquema

clients(id, rut, nombre)
web_clients(client_id)
app_clients(client_id)
atm_clients(client_id)

240 clientes con 1–3 canales asignados (web, app, ATM). Incluye casos exclusivos por canal para ejercitar diferencias.

Objetivo

Explorar: UNION, UNION ALL, INTERSECT, EXCEPT (SQLite) y sus equivalentes con JOIN en MySQL.

Contexto de negocio: migrar clientes hacia APP (canal de menor costo).

SQLite — ejemplos rápidos

-- Unión sin duplicados (web ∪ app)
SELECT client_id FROM web_clients
UNION
SELECT client_id FROM app_clients;

-- Intersección (web ∩ app)
SELECT client_id FROM web_clients
INTERSECT
SELECT client_id FROM app_clients;

-- Diferencia (web \ app)
SELECT client_id FROM web_clients
EXCEPT
SELECT client_id FROM app_clients;

MySQL — equivalentes

-- Intersección (INNER JOIN)
SELECT w.client_id
FROM web_clients w
INNER JOIN app_clients a USING (client_id);

-- Diferencia (LEFT JOIN + IS NULL)
SELECT w.client_id
FROM web_clients w
LEFT JOIN app_clients a USING (client_id)
WHERE a.client_id IS NULL;

-- Tres canales
SELECT w.client_id
FROM web_clients w
JOIN app_clients a USING (client_id)
JOIN atm_clients t USING (client_id);

Cómo abrir en DBeaver (SQLite)

  1. Descarga be_channels.sqlite.
  2. En DBeaver → Database > New Connection > SQLite.
  3. Elige el archivo descargado y conecta.

Cómo importar en phpMyAdmin (MySQL)

  1. Descarga be_channels_mysql.sql.
  2. Abre phpMyAdmin → selecciona tu base de datos.
  3. Pestaña Importar → carga el archivo y ejecuta.

Tip para ver nombres/RUT

-- SQLite / MySQL
SELECT c.rut, c.nombre
FROM clients c
JOIN app_clients a ON a.client_id = c.id
ORDER BY c.nombre;
